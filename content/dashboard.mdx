# 工作台總覽

工作台總覽是登入後的預設首頁，提供當日工作狀況的快速摘要。頁面整合了待審核分析、未讀 LINE 訊息等待處理事項，讓您一眼掌握優先工作順序。左側側邊欄為主要導覽入口，依功能類別分為五大區塊，各區塊可展開或收合。側邊欄的紅色數字徽章代表需要處理的未完成項目，是每次登入後第一個應查看的地方。

## 操作步驟

1. 登入後預設顯示「工作台總覽」，列出待審核分析、未讀訊息等摘要
2. 左側側邊欄依功能分為：病患管理、餐點審核、LINE 互動、健康數據、個人設定五大區塊
3. 點擊側邊欄的任意項目即可切換功能頁面
4. 紅色數字徽章代表待處理項目數量（如待審核數、未讀訊息數）
5. 點擊底部「返回主頁」可回到系統首頁（非工作台）
6. 點擊底部「說明文件」可開啟本頁面

## 常見問題

### 側邊欄的紅色徽章數字不準確，和實際數量不同？

徽章數字在切換頁面或重新載入時會更新。若發現數字與實際不符，可重新整理頁面（F5）或切換到其他功能再切回，觸發重新載入。

### 總覽頁面的資料會自動更新嗎？

目前總覽頁面不會自動即時更新，需手動重新整理或切換功能頁面後再切回才會更新數據。若需即時掌握新訊息，建議開啟瀏覽器桌面通知。
