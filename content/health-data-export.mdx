# 健康數據匯出

健康數據匯出功能可將病患的體重、血壓、血糖、身高等歷史健康記錄匯出為 Excel 或 PDF 檔案。支援依指標類型與時間區間篩選，匯出結果包含量測時間、數值、健康狀態分級（正常/偏高/偏低）、量測情境（空腹/餐後等）與備註。匯出的報表適合用於回診報告、跨科會診參考或病患自我管理。

## 操作步驟

1. 點擊左側「病患統計」，選擇要匯出的病患
2. 在右側詳細數據頁面中，找到下方的「健康歷史記錄」表格
3. 使用「指標篩選」下拉選單選擇要匯出的指標類型（全部 / 體重 / 血壓 / 血糖 / 身高）
4. 使用「時間篩選」下拉選單選擇時間範圍（全部 / 近 7 天 / 近 30 天 / 近 90 天）
5. 點擊「匯出 Excel」按鈕下載 .xlsx 檔案，包含篩選後的所有健康記錄
6. 或點擊「匯出 PDF」按鈕下載格式化的 PDF 報告 *(含健康狀態色彩標示：綠色=正常、紅色=偏高、橘色=偏低)*
7. 檔名格式為「病患姓名_健康數據_日期」

## 常見問題

### 匯出的數據量有上限嗎？

單次匯出最多 1,000 筆記錄，時間跨度最長 365 天。若未指定時間範圍，預設匯出近 90 天的資料。如需更長時間的數據，請分次匯出不同時間區間。

### 匯出的 Excel 包含哪些欄位？

包含：記錄時間、指標類型（體重/血壓/血糖等）、量測數值、單位、健康狀態分級、量測情境（空腹/餐後/運動前等）、備註。表頭有粗體與背景色，方便閱讀。

### PDF 報告有色彩標示嗎？

有。PDF 報告中每筆記錄會依健康狀態以色彩標示：綠色代表指標在正常範圍、紅色代表偏高、橘色代表偏低，方便快速辨識異常數值。報告底部附有記錄筆數與生成時間。

### 可以只匯出單一指標嗎？

可以。在匯出前先用「指標篩選」下拉選單選擇特定指標（如只選「血糖」），再點擊匯出按鈕，就只會匯出該指標的記錄。
